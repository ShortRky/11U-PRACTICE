📐 Formula Reference

All the formulas you need for 11U Math

📊 Functions

Linear Function

f(x) = mx + b

m = slope, b = y-intercept

Slope Formula

m = (y₂ - y₁) / (x₂ - x₁)

Rise over run

Point-Slope Form

y - y₁ = m(x - x₁)

Using a point (x₁, y₁)

Quadratic Standard

f(x) = ax² + bx + c

a ≠ 0, creates parabola

Quadratic Vertex

f(x) = a(x - h)² + k

Vertex at (h, k)

Axis of Symmetry

x = -b / (2a)

For ax² + bx + c

Quadratic Formula

x = (-b ± √(b²-4ac)) / 2a

For solving ax² + bx + c = 0

Exponential Function

f(x) = a · bˣ

a = initial value, b = base

Exponential Growth

A = P(1 + r)ᵗ

P = principal, r = rate, t = time

Exponential Decay

A = P(1 - r)ᵗ

For decay problems

📐 Trigonometry

SOH (Sine)

sin θ = Opp / Hyp

Sine = Opposite / Hypotenuse

CAH (Cosine)

cos θ = Adj / Hyp

Cosine = Adjacent / Hypotenuse

TOA (Tangent)

tan θ = Opp / Adj

Tangent = Opposite / Adjacent

Pythagorean Identity

sin²θ + cos²θ = 1

Fundamental trig identity

Pythagorean Theorem

a² + b² = c²

For right triangles

Law of Sines

a/sin A = b/sin B = c/sin C

For any triangle

Law of Cosines

c² = a² + b² - 2ab cos C

Generalized Pythagorean

📋 Sequences & Series

Arithmetic Sequence

aₙ = a₁ + (n-1)d

nth term of arithmetic sequence

Arithmetic Series Sum

Sₙ = n(a₁ + aₙ) / 2

Sum of n terms

Arithmetic Series (alt)

Sₙ = n[2a₁ + (n-1)d] / 2

Using first term and d

Geometric Sequence

aₙ = a₁ · rⁿ⁻¹

nth term of geometric sequence

Geometric Series Sum

Sₙ = a₁(1 - rⁿ) / (1 - r)

For r ≠ 1

Infinite Geometric Series

S∞ = a₁ / (1 - r)

Only if |r| < 1
